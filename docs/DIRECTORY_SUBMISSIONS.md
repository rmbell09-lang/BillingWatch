# BillingWatch — SaaS Directory Submission Guide

## Product Info (copy-paste ready)
- **Name:** BillingWatch
- **Tagline:** Catch billing anomalies before your customers do
- **URL:** https://billingwatch.pages.dev (pending deploy)
- **Category:** Developer Tools / Billing / SaaS / API
- **Pricing:** Free beta (open source core)
- **Description (short):** AI-powered billing anomaly detection for Stripe. 10 detectors catch phantom charges, price drift, usage spikes, duplicate invoices, and more. Self-hosted, privacy-first, real-time alerts.
- **Description (long):** BillingWatch monitors your Stripe billing events in real-time using 10 specialized anomaly detectors. It catches issues like sudden price changes, duplicate charges, phantom subscriptions, unusual usage spikes, and failed payment clusters — before your customers complain or you lose revenue. Self-hosted on your infrastructure, zero data leaves your servers. REST API with webhook alerts, email notifications, and a visual dashboard.

## Key Features (bullet points)
- 10 anomaly detectors (phantom charges, price drift, duplicate invoices, usage spikes, etc.)
- Real-time Stripe webhook processing
- Visual dashboard with Chart.js (dark mode, auto-refresh)
- Email/Slack/Discord alert channels
- REST API for integration
- Self-hosted — your data stays on your servers
- SQLite — zero external dependencies
- False positive marking with per-detector tracking

## Target Audience
- SaaS founders with Stripe billing
- DevOps/billing engineers
- Finance teams at software companies
- Anyone who has been burned by billing bugs
