- Visualizations:

    - Doughnut chart: alert severity breakdown (critical/high/medium/low)
    - Latest alert panel with detector name, severity, timestamp
- Fetches data from /dashboard/summary + /metrics/detectors via JS
- Status badge (healthy / warning / critical / no_data) in header
- Color-coded failure rate (green < 5%, yellow 5-15%, red > 15%)
- Existing /dashboard/summary JSON endpoint preserved
