FAQ

Common questions

Do I need to change my Stripe setup?

Just add a webhook endpoint. That's it. No code changes, no SDK, no infrastructure.

What if I already use Stripe Radar?

Radar catches fraud at the authorization level. BillingWatch catches anomalies in your billing patterns — stuff Radar doesn't see, like silent subscription lapses, duplicate charges, or webhook pipeline lag.

Is my Stripe data safe?

We only receive webhook event payloads — the same data Stripe sends to any endpoint. We never store card numbers or customer PII beyond what's in the event.

What if I want a detector you don't have?

Pro and Scale plans support custom detector rules. Tell us what billing edge case keeps you up at night — we'll build it.
