7 Real-Time Anomaly Detectors

DetectorWhat It CatchesSeverity
charge_failure_spikeFailure rate >15% over 1 hour — gateway issue or card BIN attack🔴 High
duplicate_chargeSame customer charged twice within 5 min — idempotency bug🚨 Critical
revenue_dropMRR drops >20% day-over-day — silent cancellation or webhook backlog🔴 High
refund_spikeRefund rate >10% — product or fulfillment issue🟡 Medium
dunning_failureRepeated retry failures — card expired, not updated🔴 High
fraud_spikeDisputed charge rate rises — carding attack in progress🚨 Critical
subscription_lapseSubscriptions silently lapsing without cancellation event🟡 Medium

One-Line Install

$ git clone https://github.com/rmbell09-lang/BillingWatch
$ cd BillingWatch
$ pip install -r requirements.txt
$ cp .env.example .env  # add your Stripe key
$ python main.py

Runs on Python 3.9+. No Docker required. No cloud dependency.

No SaaS fees. No data leaks. Just alerts.
